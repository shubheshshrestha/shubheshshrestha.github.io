

# Portfolio Website

This repository hosts my personal portfolio website, built with an HTML5 UP template and customized to showcase my work in **Data Analytics, Django/Backend Development, and Graphic Design**.  

The website is deployed via **GitHub Pages** and serves as a central hub for my projects, designs, and professional background.  

## 🌐 Live Site
👉 [shubheshshrestha.github.io](https://shubheshshrestha.github.io)  
*(or custom domain if connected later)*

## 📂 Sections
- **Data Analytics Projects** — Excel, Power BI, Tableau, MySQL, and Pandas projects with repo links and dashboards.  
- **Django / Backend Projects** — Web apps and backend systems built with Django and REST APIs.  
- **Graphic Designs** — Posters, product creatives, and marketing designs.  
- **About Me** — Education, skills, and career goals.  
- **Contact** — Ways to connect (LinkedIn, GitHub, Email).  

## ⚙️ Tech Stack
- HTML5, CSS3, JavaScript  
- Responsive layout (HTML5 UP template customization)  
- GitHub Pages for hosting  

## 🚀 Deployment
The site is automatically deployed from the `main` branch via **GitHub Pages**.  
Any changes pushed to this branch are reflected live.
